Event 6: LinkedIn: Git Essential Training

Date: 16/06/2025

Summary

This was the last LinkedIn Learning module that I completed. It was beginner training for Git, the distributed version control system used throughout the industry. This is something that I’ve never had any formal training in throughout my degree. This course covered quite a lot, first introducing the motivations of why we need version control (explaining the example that I’m sure every single programmer can relate to, where you have some code in a file, and then you copy it and rename it so you can test something or add something else, and then you do it again, and again, and again…) and how Git came to be. It taught all the fundamental skills and I feel has put me in a good position to feel confident using Git going forward.

Additional Comments

The course also taught how to install and configure git, setting up a repository, ‘staging’ and committing a file, push and pulling, branches (and pull requests) and merges, and merge conflicts. Merge conflicts have been my biggest issue using Git at Uni, and it was interesting to see a professional’s take on how to resolve them. I wasn’t expecting completely deleting everything in the repo and starting again from the local as an actual professional solution!
